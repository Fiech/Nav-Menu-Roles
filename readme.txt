== Description ==

This plugin lets you hide custom menu items based on user roles.  


== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance > Menus
1. Edit the menu items accordingly.  Check the boxes next to the roles you'd like to restrict visibility to.
1. If you don't check any boxes, the item will be visible to everyone like normal.
